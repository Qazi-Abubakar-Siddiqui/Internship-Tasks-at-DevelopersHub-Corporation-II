# Main Repository README.md

# AI & Machine Learning Business Analytics Projects

## Overview

This repository contains five complete AI, Machine Learning, Data Science, and Business Analytics projects implemented using Python. The projects focus on supervised learning, unsupervised learning, time-series forecasting, business optimization, and dashboard development.

Each project includes:

* Complete data preprocessing
* Exploratory Data Analysis (EDA)
* Machine Learning models
* Model evaluation
* Visualizations and charts
* Business insights and conclusions
* Downloadable outputs
* Google Colab compatible notebooks

---

# Projects Included

## 1. Bank Marketing Term Deposit Prediction

### Objective

Predict whether a customer will subscribe to a term deposit using classification algorithms.

### Algorithms Used

* Logistic Regression
* Random Forest
* SHAP Explainability
* ROC Analysis

### Key Features

* Feature Encoding
* Model Comparison
* Confusion Matrix
* ROC Curve
* SHAP Explanations

---

## 2. Customer Segmentation Using K-Means

### Objective

Segment mall customers based on income and spending behavior.

### Algorithms Used

* K-Means Clustering
* PCA
* t-SNE

### Key Features

* Cluster Visualization
* Customer Insights
* Marketing Recommendations

---

## 3. Energy Consumption Forecasting

### Objective

Forecast household power consumption using time-series models.

### Algorithms Used

* ARIMA
* Prophet
* XGBoost

### Key Features

* Time-Series Resampling
* Feature Engineering
* Forecast Comparison
* Actual vs Forecast Visualizations

---

## 4. Loan Default Risk Prediction

### Objective

Predict loan default risk and optimize business costs.

### Algorithms Used

* Logistic Regression
* CatBoost

### Key Features

* Cost Optimization
* Threshold Tuning
* Risk Analysis
* Business Decision Modeling

---

## 5. Interactive Streamlit Dashboard

### Objective

Develop an interactive dashboard for business analytics.

### Features

* KPI Monitoring
* Sales Analysis
* Profit Analysis
* Region Filters
* Category Filters
* Top Customer Analysis
